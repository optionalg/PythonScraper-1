The Beginning¶

After looking through an exhausting amountof public datasets I finally stumbled upon a set of nearly all IGN's game ratings at https://www.dropbox.com/s/09sh15zbtwg9eu3/gamedata.xlsx. Not only was this a of a more managable size than every near-earth asteroid in the solar system (hint: theres alot) but I know something about IGN ratings. I wondered: how do IGN ratings compare to that of another site, such as Metacritic? Now, I had a starting point!

I set to learning Python in order to create a web scraper to get metacritic scores for the IGN games I had. After several hours, I (and many more of formatting) I completed my scraper. You can find the code for it here: https://github.com/wraley/PythonScraper

My scraper did its job and found metacritic scores for a significant portion of my database. Sadly, Metacritic does not have quite as extensive a database as IGN so our new dataset is much smaller. The next step is to clean up some of our results.

Cleaning up¶

We'll use the pandas module for manipulating out data, and matplotlib for visualizing the data in some neat boxplots.

The first step is to start cleaning the results from my metacritic scraper. Unfortunately, due to the nature of my code the are far from beautiful. Xlrd gives uses commas to break up cells, and so game titles with commas got butchered. Some other character combinations or other errors resulted in a designated 'failed' status. Lastly, metacritic does not have as extensive a game database as IGN so many of the games did not have a rating. We'll need to remove any rows that fit this criteria.

In [71]:
from matplotlib import pyplot as plt
import pandas as pd
plt.style.use('ggplot')
import numpy as np

df = pd.read_csv('E:\Data Libraries\Billy\coding\metacriticScraper\Final_Data.csv')

# Remove any rows where scraper failed to obtain score, or where Metacritic did not have a rating
df = df.drop(df[df.Meta_Rating == 'failed'].index)
df = df.drop(df[df.Meta_Rating == 'None'].index)

Polishing off¶

Now that we've removed our trouble rows, we can go about fixing our data types. As I said before, some of the rows got butchered, meaning there were strings in ratings columns. We've successfully dealt with this so we can set about casting our columns into their proper datatypes for manipulation later on. Last but not least, we'll scale the IGN Ratings by 10 so that its on the same scale as Metacritic.

In [72]:
# Now that all the columns agree, we can cast the ratings columns to floats
df['IGN_Rating'] = df['IGN_Rating'].astype('float64')
df['Meta_Rating'] = df['Meta_Rating'].astype('float64')

# IGN rates from 0-10 and Metacritic rates from 0-100 so lets multiply IGN by 10 for simplicity
df['IGN_Rating'] *= 10

Stats¶

Coming into this project, I expected to find a large disparity between my two datasets. I regularly read IGN reviews, and pay attention to their ratings. Many games, particularly AAA titles like Call of Duty rarely get below a 90-95 despite fans feeling otherwise. I've also heard stories basing many of their financial decisions on metacritic ratings. As a result, I expected to find IGN ratings skewed towards the left, and Metacritic to have a more even distribution.

As it turns out, my assumptions were quite wrong. The means, standard deviation. as well as our 3 intervals we all within 4 points of each other. It seems that the game ratings community is suprisingly consistent.

The matplotlib module also has some handy visualization methods that help show just how consistent these two sites are. Both are skewed right of a perfect bell curve, but I suppose that means people have been making some pretty good games. Alternatively, (and more likely) IGN and Metacritic dont have reviews for games that are complete shit.

The results¶

The results for the Call of Duty dataset are a little less clear. The first thing I noticed is that there is a clear outlier in the 30-40 range. Upon furhter inspection, its seems that Call of Duty: Black Ops Declassified for the PSP was very unpopular. Next, the histogram shows that I was at least partially right in that IGN is more likely to give a Call of Duty game a 90+ rating, yet overall they are more inconsistent, particularly with older titles. metacritic has a tighter distribution, yet IGn and Metacritic average out at 81 and 79, respectively.
